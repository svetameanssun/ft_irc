## Description

This project aims to provide an understanding of how the TCP/IP layer works by using the UNIX socket API to create a functional IRC server compatible with official IRC clients.

## Instructions

To compile the project, a Makefile is provided.  
Use the `make` command to compile it.

To run the project, several commands are available:

- `make run`: executes the server with predefined `<port>` and `<password>`
- `make valgrind`: executes the server and logs Valgrind checks to detect memory errors and/or open file descriptors
- `make valgrind_logs`: logs both Valgrind errors **and** the custom logs generated by the server, depending on the operations performed

## Resources

As a reference for the implementation of the IRC server, we have almost exclusively used the RFCs related to the IRC protocol, such as:

- RFC 1459 (May 1993)
- RFC 2811 (April 2000)
- RFC 2812 (April 2000)

For the networking part, several YouTube resources were used to understand the UNIX networking API and its underlying concepts.
